1, 执行quickstart.bat脚本
	1）install插件jar到本地maven仓库
	mvn install:install-file // 通过install的方式引入二方库到本地仓库
		-DgroupId=quickstart 
		-DartifactId=web 
		-Dversion=1.0.0 
		-Dpackaging=jar 
		-Dfile=quickstart-web.1.0.0.jar  // 本地Jar文件
		-DgeneratePom=true


	2）通过插件archetype方式，构建一个SpringMVC的web项目
	
	mvn archetype:create 	// archetype插件命令
		-DarchetypeGroupId=quickstart // 同前面install的groupId
		-DarchetypeArtifactId=web 	  // 同前面install的artifactId
		-DgroupId=com.yunos 		  // 指定当前新建的Project的groupId
		-DartifactId=quickstart		  // 指定当前新建的Project的artifactId
2, Eclipse新建一个普通的Java项目，Import刚生成的目录到当前项目，并把该项目转成Maven Project:
	右键选中Project>>Configure>>Configure to Maven Project
